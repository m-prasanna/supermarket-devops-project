# 🛒 Smart Super Market DevOps Project

A production-ready Super Market Web Application deployed using DevOps best practices with Docker, Nginx, GitHub Actions CI/CD, and AWS EC2.

This project demonstrates practical DevOps skills for portfolio, GitHub, and CV purposes.

---

# 🚀 Project Overview

This project is designed to showcase:

- Linux Server Administration
- Docker Containerization
- Docker Compose Orchestration
- Git & GitHub Version Control
- GitHub Actions CI/CD Pipeline
- AWS EC2 Deployment
- Nginx Reverse Proxy
- Node.js Backend Deployment
- HTML/CSS/JavaScript Frontend Hosting
- Production-ready Deployment Workflow

---
